    (이 파일을 관리자 권한으로 열어 ip주소에 해당하는 주소 이름을 지정해보자)
Domain Server DNS : 관련 url의 ip주소를 가지고 있는 서버

ip의 종류
    공인 IP (Public 2p) -> 인터넷 상에서 유일
        고정 IP : 바뀌지 않는 IP. 
        동적 IP : 바뀌는 IP. 돌아가면서 사용하는 IP
    사설 IP (Private 2P) -> 마구 만드는 것

DHCP : Dynamic Host Configuration Protocol
공유기는 공인IP이며 여러 인터넷, PC, 휴대폰 등을 연결해준다.

SI : 시스템 통합. 관공서 같은 다양한 시스템을 통합한 것들을 예로 들 수 있다.
